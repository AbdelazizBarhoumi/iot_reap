title IoT-REAP - Teacher Forum Moderation Sequence Diagram
actor Teacher as teacher
participant "Frontend\n(React/Inertia)" as frontend

participant "Forum\nController" as forum

teacher -> frontend : Navigate to teaching forum inbox
frontend -> routes : GET /teaching/forum/inbox
routes -> forum : teacherInbox()
forum -> db : Load teacher's training paths
forum -> db : Load threads for those paths
forum -> db : Load recent replies
db --> forum : Return forum data
forum --> routes : Return inbox data
routes --> frontend : Display forum inbox
frontend --> teacher : Show threads and replies for teacher's courses

teacher -> frontend : View thread details
frontend -> routes : GET /forum/threads/{threadId}
routes -> forum : show(threadId)
forum -> db : Load thread with replies
forum -> db : Load upvotes and flags
db --> forum : Return thread data
forum --> routes : Return thread
routes --> frontend : Display thread
frontend --> teacher : Show thread with all replies

teacher -> frontend : Pin thread
frontend -> routes : POST /forum/threads/{threadId}/pin
routes -> forum : pin(threadId)
forum -> db : Update thread pinned status
forum -> db : Set pinned_at timestamp
forum -> db : Save changes
forum --> routes : Return success
routes --> frontend : Update thread display
frontend --> teacher : Show thread as pinned

teacher -> frontend : Unpin thread
frontend -> routes : POST /forum/threads/{threadId}/unpin
routes -> forum : unpin(threadId)
forum -> db : Update thread pinned status
forum -> db : Clear pinned_at timestamp
forum -> db : Save changes
forum --> routes : Return success
routes --> frontend : Update thread display
frontend --> teacher : Show thread as unpinned

teacher -> frontend : Lock thread

note right of teacher
  Teacher forum moderation features:
  - View inbox for their training paths
  - Pin/unpin important threads
  - Lock/unlock threads to prevent new replies
  - Mark replies as accepted answers
  - Resolve flags on content
  - Post replies and upvote content
  - Flag inappropriate content for admin review
  - Delete own content
  - Rate limited write operations (throttle:forum-write)
end note
