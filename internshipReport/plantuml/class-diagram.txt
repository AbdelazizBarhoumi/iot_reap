note right of User
    **Role System**: Simple string-based
    No separate Role/Permission models
    Three hardcoded roles:
    - superadmin
    - department_admin  
    - employer
end note

note bottom of DigitalItem
    **Pivot Table**: novation_inov_siri_submission_digital_item
    Contains field_type: 'tool' or 'gap'
    Used to distinguish deployed tools from identified gaps
end note

note top of SiriAIService
    **Google Gemini AI Integration**
    - 60s timeout, 10s connection timeout
    - Input sanitization for security
    - URL normalization
    - Maturity level progression logic
end note

note bottom of UpdateJob
    **System Update Management**
    Tracks application version updates
    Stores backup information
    Progress logging and error handling
end note

note right of DatabaseNotification
    **Laravel Notification System**
    Polymorphic relationship with User via Notifiable trait
    Stores in-app notifications:
    - Account status changes
    - Appeal decisions
    - System updates
    - Admin actions
end note
